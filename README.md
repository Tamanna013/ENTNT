# 🚢 FleetMind AI

**Status: 🟡 In Development**

---

FleetMind AI is an enterprise-grade, AI-powered Maritime Operations Management Platform designed for shipping companies to efficiently manage their fleets, crews, voyages, cargo, maintenance schedules, and regulatory compliance — all from a single unified dashboard enhanced with intelligent, AI-assisted insights and predictive analytics.

---

## Tech Stack

| Layer        | Technology                                          |
| ------------ | --------------------------------------------------- |
| **Frontend** | React · TypeScript · Tailwind CSS · TanStack Query · Zustand · Recharts |
| **Backend**  | ASP.NET Core 9 · Entity Framework Core · SQL Server |
| **Auth**     | JWT with Refresh Tokens                             |
| **AI**       | Azure OpenAI · Gemini                               |
